Deployment Steps

1
Push to GitHub

Initialize your local repository and push all generated HTML files to your main branch.

2
Enable Pages

Go to Repository Settings > Pages. Select the "main" branch and "/ (root)" folder then click Save.

3
Verify Links

Wait a few minutes for the build to finish. Your hub will be live at username.github.io/repo-name/.

info
